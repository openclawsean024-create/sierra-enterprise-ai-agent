# Sierra Enterprise AI Agent

企業客服 AI Agent，部署於網站或通訊平台，全天候處理客戶常見問題、引導用戶操作、並在必要時無縫轉接真人客服。

## Tech Stack
- Next.js 15 + React 19 + TypeScript
- GPT-4o mini for NLU and response generation
- Vercel deployment

## Features
- FAQ Bot with NLU intent recognition
- Multi-turn dialogue management
- Human handoff mechanism
- Ticket system integration
- Multi-language (繁體中文, English, 簡體中文)
- Analytics dashboard
- Chat widget embed
- Webhook API

## Getting Started
```bash
npm install
npm run dev
```
